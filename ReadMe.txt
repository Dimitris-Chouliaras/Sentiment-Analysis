Το πρόγραμμα είναι γραμμένο σε γλώσσα προγραμματισμού Python, σε περιβάλλον PyCharm.

Αφού γίνουν τα κατάλληλα imports στην αρχή του προγράμματος και τα κατάλληλα downloads, μέσω terminal μπορούμε να τρέξουμε το πρόγραμμα με την εντολή: streamlit run main.py

Το excel αρχείο περιλαμβάνει κάποιες προτάσεις στις οποίες πραγματοποιήθηκε η ανάλυση συναισθήματος και βγήκαν τα αποτελέσματα.
